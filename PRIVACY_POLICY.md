# Privacy Policy

Effective date: 2026-07-06

This Privacy Policy explains how **B站同源助手 for YouTube™** ("the extension") collects, uses, stores, and shares data.

## Summary

The extension helps users find B站 videos that may match or relate to the current YouTube™ video page.

The extension does not operate a developer-owned backend server, does not sell user data, and does not use user data for advertising or tracking.

## Data the extension handles

When you open a YouTube™ video page, the extension may read:

- Current YouTube™ video title
- Channel name displayed on the current YouTube™ page
- Video duration
- YouTube™ video ID
- Current YouTube™ page URL
- Your explicit extension actions, such as opening a B站 result, using "like and open", disabling automatic search, or clearing local records

The extension uses this data only to provide its single purpose: searching for relevant public B站 results, ranking those results locally, showing the helper UI on the current page, and remembering user-confirmed matches in your browser.

## Data stored locally

The extension may store the following data in Chrome local storage on your device:

- Extension settings
- Search cache
- User-confirmed YouTube™ to B站 match records

This local data stays in your browser. It is not uploaded to a developer-owned server.

You can delete local extension data by using the clear option in the extension popup, clearing Chrome site or extension storage, or removing the extension from Chrome.

## Data shared with third parties

To find B站 results, the extension may send a search query derived from the current YouTube™ video title to public B站 search services, including:

- `https://api.bilibili.com/`
- `https://search.bilibili.com/`
- `https://www.bilibili.com/`

The query is used to request public B站 search results. The extension does not send this data to a developer-owned server.

If you choose the "like and open" action, the extension clicks the existing YouTube™ like button on the current page before opening the B站 result. That action is handled by YouTube™ under your relationship with YouTube/Google.

## Data not collected

The extension does not intentionally collect:

- Personally identifiable information such as name, email address, phone number, or government ID
- Financial or payment information
- Health information
- Authentication information such as passwords, tokens, or cookies
- Personal communications
- Precise location

## Remote code

The extension does not download or execute remote code. All extension logic is included in the published extension package. Remote requests are used only to fetch public B站 search data or pages.

## User controls

You can:

- Disable automatic search in the extension popup
- Clear stored match records in the extension popup
- Remove the extension from Chrome at any time

## Changes to this policy

If this policy changes, the effective date at the top of this page will be updated.
