# B站同源助手 for YouTube™

一个原生 Chrome MV3 插件。打开 YouTube 视频页时，它会在右侧推荐区顶部自动搜索 B站同源或相近视频，找到后提示你一键打开。

## 第一版能力

- 自动识别当前 YouTube 视频标题、频道名和时长。
- 自动搜索 B站视频结果。
- 用标题相似度、时长误差、UP 主信息和播放量做本地评分。
- 在右侧推荐区顶部提示，不挤占视频标题、描述和评论区。
- 区分 `很像同一个` 和 `可能相关`，低置信结果保留手动 B站搜索入口。
- 高置信结果支持一键打开 B站、点赞打开。
- 打开 B站或 B站搜索前会自动暂停当前 YouTube 视频，避免后台继续播放。
- 可能相关结果只提供打开看看、去 B站搜，避免误点赞。
- 用户确认过的 YouTube 到 B站映射只保存在本机。
- B站 API 搜索被风控时，自动用真实 B站搜索页读取候选结果。

## 安装

1. 打开 Chrome 的 `chrome://extensions/`。
2. 打开右上角 `开发者模式`。
3. 点击 `加载已解压的扩展程序`。
4. 选择这个目录：`yt-bili-helper`。
5. 打开任意 YouTube 视频页测试。

也可以下载 GitHub Releases 里的 `yt-bili-helper-*.zip`，解压后按上面的方式加载。

## 使用

打开 YouTube 视频页后，右侧推荐列表顶部会出现轻量提示：

- `打开` / `打开看看`：先暂停当前 YouTube 视频，再在新标签打开匹配到的 B站视频，并记住高置信选择。
- `点赞打开`：先尝试给 YouTube 原视频点赞并暂停当前视频，再打开 B站。
- `去 B站搜`：低置信或未找到时，直接打开 B站搜索结果页。

## 重要说明

- 插件不伪造 YouTube 内部上报请求；点赞使用当前页面已有的 YouTube 点赞按钮。
- 如果你的 VPN 或代理是全局模式，打开 B站也可能走代理。想省流量，需要让 `bilibili.com`、`hdslb.com`、`bilivideo.com` 走直连或国内线路。
- B站搜索接口有时会返回风控或验证页。插件会自动尝试网页搜索兜底；如果仍失败，会提供手动搜索按钮。

## 隐私

插件没有开发者自有后端，不出售用户数据。搜索 B站结果时，插件会把从当前 YouTube 视频标题生成的搜索词发送到公开 B站搜索端点或搜索页面。详见 [PRIVACY_POLICY.md](PRIVACY_POLICY.md)。
